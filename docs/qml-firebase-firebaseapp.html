Detailed Description

The FirebaseApp type holds crucial data from the Firebase project such as API Key or database URL which will then be passed on to FirebaseAuth or FirebaseDatabase. The properties of the Firebase App can be set by downloading the google-services.json file from Firebase and store it in the resources for automatic parsing. An alternative is manually setting the properties in qml:

FirebaseApp {
    id: fbApp
    apiKey: "insert-key"
    databaseUrl: "insert-url"
}

FirebaseDatabase {
    apiKey: fbApp.apiKey
    databaseUrl: fbApp.databaseUrl
}

See also FirebaseAuth, FirebaseDatabase, and FirebaseUser.

Property Documentation

apiKey : string

This property holds the API Key of the Firebase project, used by the types FirebaseDatabase and FirebaseAuth.


authDomain : string

Domain of the authentication, currently not used.


databaseUrl : string

Property that holds the URL where the Firebase Realtime Database resides and should be passed to FirebaseDatabase objects.


source : url

This property holds the URL of the google-services.json that contains all information about a Firebase Project (usually obtained in Firebase Console). If this path is set, the file will be parsed and all the other properties will be automatically set.


storageBucket : string

This property holds the URL of the Firebase Storage, currently not implemented.
