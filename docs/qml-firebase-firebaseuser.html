Detailed Description

FirebaseUser simply holds data about the users authenticated in the firebase. This type is used by FirebaseAuth to hold a reference to the user currently authenticated and can also be used to create models/lists of all available users in the FirebaseDatabase.

Property Documentation

birthdate : string

Optional field about the user's birthdate.


email : string

Specifies the user's email.


emailVerified : bool

Specifies if the email of this user has already been verified. A verification email can be sent using FirebaseAuth::sendEmailVerification()


idToken : string

Specifies the users idToken, often used for actions that require authentication such as writing/reading data from the FirebaseDatabase or managing the user profile with FirebaseAuth. This token expires after 1 hour and should be refreshed using the refreshToken.

Note: for more information about use, see FirebaseAuth::currentUser


name : string

Returns the name of the user.


photoUrl : string

Url that holds the profile picture of the user.


profession : string

Optional field about the user's profession.


refreshToken : string

Returns the refresh token, tipically used to refresh the idToken after it expires.

Note: for more information about use, see FirebaseAuth::exchangeRefreshToken


userId : string

Specifies the unique identifier of the user in the entire Firebase project.

Warning: this identifier will change if the email of the user is altered.
