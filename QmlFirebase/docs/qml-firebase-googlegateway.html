Detailed Description

This type allows the user to authenticate on the firebase using a google account. By calling the fetchGoogleToken() a browser window will be opened and the user can log in. After the login is complete, if successful an authToken will be retrieved which can be used with FirebaseAuth::signInWithOAuthCredential. See example below:

FirebaseApp {
    id: fbApp
    apiKey: "insert-key"
    databaseUrl: "insert-url"
}

FirebaseAuth {
    id: fbAuth
    apiKey: fbApp.apiKey
    onSignedIn: ...
}

GoogleSignInHelper {
    id: googleSignIn

    onAuthTokenChanged: {
        console.log("Token granted:  \n" + authToken)
        fbAuth.signInWithOAuthCredential(authToken, "google.com")
    }
}

Button {
    text: "Google login"
    onClicked: googleSignIn.fetchGoogleToken()
}

See also FirebaseAuth and FirebaseApp.
